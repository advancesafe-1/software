Place Mosquitto Windows 64-bit binaries here for bundling with the installer:
- mosquitto.exe
- mosquitto.conf
- mosquitto_passwd.exe
- libssl-3-x64.dll, libcrypto-3-x64.dll (and any other DLLs from the package)

Download from https://mosquitto.org/download/
If this folder is empty, the installer will skip MQTT service installation.
